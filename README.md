# Projet ML Retail

## Description du projet
Ce projet met en place une structure complète pour un workflow de Machine Learning appliqué au domaine retail.  
Il couvre les étapes principales : préparation des données, entraînement d’un modèle, génération de prédictions, et base de déploiement via une application Flask.
